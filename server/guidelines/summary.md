# Summary Prompt Guidelines

- Follow the general OpenAI-derived prompt rules: clear task first, useful context next, explicit output format and constraints.
- Define the source material, audience, and intended use of the summary.
- Specify the summary length, format, and level of detail.
- Ask the model to preserve key facts, decisions, dates, names, risks, and action items.
- Require separation between source-backed facts and inferred interpretation.
- Request a brief section for open questions when the source is incomplete.
