# General Prompt Guidelines

- State the exact goal and expected outcome.
- Include the relevant background, audience, domain, and current situation.
- Specify the desired output format, such as bullets, table, JSON, Markdown, code, or step-by-step explanation.
- Add constraints such as length, tone, language, deadline, tools, exclusions, or quality criteria.
- Provide examples, references, source text, or acceptance criteria when available.
- Ask the model to call out assumptions when important information is missing.
