# Coding Prompt Guidelines

- Follow the general OpenAI-derived prompt rules: clear task first, useful context next, explicit output format and constraints.
- Describe the project stack, runtime, framework, package manager, and relevant file paths.
- State the expected behavior, current behavior, errors, and reproduction steps.
- Include constraints such as compatibility, performance, security, and style conventions.
- Ask for a focused implementation plan, code changes, and verification steps.
- Provide logs, snippets, API contracts, schemas, or tests when available.
