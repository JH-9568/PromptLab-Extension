# PromptLab Privacy Policy

Last updated: 2026-05-13

PromptLab helps users improve prompts on ChatGPT and collects limited anonymous usage metadata to evaluate prompt improvement quality.

## Data We Collect

PromptLab may collect:

- Anonymous user ID generated by the extension
- Session ID
- Task category
- Whether the improved prompt or original prompt was used
- Satisfaction rating selected by the user
- Before/after prompt analysis metadata
- Guideline metadata used for improvement
- SHA-256 hashes of the original and improved prompts
- Character lengths of the original and improved prompts
- Server request metadata that may be generated by hosting infrastructure

## Data We Do Not Store

PromptLab does not intentionally store:

- Full original prompt text
- Full improved prompt text
- ChatGPT response text
- Names, emails, phone numbers, passwords, payment data, or other directly identifying personal information

Users should avoid entering sensitive personal information into prompts.

## How We Use Data

Collected metadata is used to:

- Measure whether prompt improvements are useful
- Compare prompt specificity before and after improvement
- Debug and improve PromptLab functionality
- Export aggregate or session-level research data

## Data Sharing

PromptLab sends prompt text to the backend server to generate an improved prompt. The backend may send prompt text to OpenAI API for prompt rewriting. Prompt text is not stored by PromptLab logs.

Session metadata is stored in Supabase. The backend is hosted on Render.

## Security

PromptLab uses HTTPS for communication with the backend server. API keys and database service keys are stored only in backend server environment variables and are not included in the Chrome extension.

## Data Retention and Deletion

PromptLab keeps anonymous session metadata for research and product improvement. To request deletion of collected data associated with a specific anonymous user ID, contact the project maintainer with that anonymous user ID.

## Contact

For questions about this policy, contact the project maintainer through the GitHub repository.
